# 📁 目录结构重构执行计划

## 📊 现状分析

### 当前问题目录
```
src/
├── 核心功能分散: engine/, executor/, collector/
├── UI重复实现: ui/, views/, gui/, viewmodels/
```

### 重构后目标结构
```
src/
├── core/           # 统一核心业务逻辑
│   ├── analyzers/      # 游戏分析器（已有unified_game_analyzer.py）
│   ├── engines/        # 决策引擎（从engine/迁移）
│   ├── executors/      # 执行器（从executor/迁移）
│   ├── collectors/     # 收集器（从collector/迁移）
│   ├── models/         # 业务数据模型（从models/部分迁移）
│   └── automation/     # 自动化逻辑（从games/迁移）
├── services/       # 基础服务层（保持现状，已重构）
├── ui/            # 统一用户界面
│   ├── windows/        # 主窗口（合并多个main_window.py）
│   ├── widgets/        # 控件（从views/widgets/迁移）
│   ├── panels/         # 面板（从views/panels/迁移）
│   ├── components/     # 组件（从views/components/迁移）
│   ├── managers/       # UI管理器（从views/managers/迁移）
│   ├── viewmodels/     # 视图模型（从viewmodels/和models/部分迁移）
│   ├── styles/         # 样式（从views/styles.py迁移）
│   └── editors/        # 编辑器（从editor/迁移）
├── utils/         # 工具函数（保持并扩展）
│   ├── performance/    # 性能工具（从performance/迁移）
│   ├── ocr/           # OCR功能（从onnxocr/迁移）
│   ├── macro/         # 宏功能（从macro/迁移）
│   └── recorders/     # 录制工具（operation_recorder.py迁移）
├── common/        # 通用组件（保持现状，已重构）
├── legacy/        # 备份实现（保持现状，已重构）
├── resources/     # 资源文件（保持现状）
└── tests/         # 测试代码（新建）
```
